# Example Obsidian Blog Post

This is a template demonstrating all the Obsidian syntax features supported by the blog pipeline.

## Frontmatter Template

Use this frontmatter in your Obsidian posts:

```yaml
---
title: "Your Post Title"
date: "2025-12-23"
description: "Brief summary of your post"
tags: ["tag1", "tag2", "tag3"]
slug: "custom-url-slug"
draft: false
---
```

**Field mappings:**
- `date` â†’ converts to `publishedAt`
- `description` â†’ converts to `summary`
- `tags` array â†’ converts to comma-separated string
- `draft: true` â†’ excludes post from publishing

## Wikilinks

Link to other blog posts using Obsidian's wikilink syntax:

- Basic link: [ict-hackathon](/blog/ict-hackathon)
- With display text: [My Instagram Bot](/blog/instagram-pfp)

**In Obsidian, write:**
```
[ict-hackathon](/blog/ict-hackathon)
[My Instagram Bot](/blog/instagram-pfp)
```

## Images

Place images in `Blog/assets/` folder in your vault.

**In Obsidian, write:**
```
![my-image](/blog-images/example-obsidian-post-my-image.png)
![Alt text for the image](/blog-images/example-obsidian-post-screenshot.jpg)
```

Images are automatically:
1. Copied to `public/blog-images/`
2. Prefixed with the post slug to prevent conflicts
3. Converted to standard markdown syntax
